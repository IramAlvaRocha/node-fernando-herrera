# Node.js — Fernando Herrera

Repositorio de ejercicios del curso **Node.js** de Fernando Herrera.

Cada carpeta numerada corresponde a una sección del curso. A medida que avances, agrega nuevas carpetas (`03-...`, `04-...`, etc.) en la raíz de este repositorio.

## Estructura

| Carpeta | Descripción |
|---------|-------------|
| `01-bases` | Fundamentos de Node.js |
| `02-multiplication` | Tabla de multiplicar con TypeScript, yargs y arquitectura por capas |

## Requisitos

- [Node.js](https://nodejs.org/) (v18 o superior recomendado)
- npm

## Cómo usar un ejercicio

Entra a la carpeta del ejercicio, instala dependencias y ejecuta los scripts definidos en su `package.json`:

```bash
cd 01-bases
npm install
npm run dev
```

```bash
cd 02-multiplication
npm install
npm run dev -- --base 5 --limit 10 --show
```
